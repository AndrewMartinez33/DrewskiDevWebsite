Face Detection App

The Face Detection app is built with React for the front-end UI, Node + Express for the backend, and Postgres for the database. The application leverages the Clarafai face detection API to place a blue box around the faces in a photograph. Users are able to register their own accounts and sign in to use the application. I also used an NPM package called bcrypt for password encryption.

  • View Github
  • Coming March 2019
  • React, Node, JavaScript, HTML, CSS, Clarifai

2018 SAS Analytics Competition

My team used a publicly available dataset from the General Social Survey. The dataset contains 5,881 attributes and 62,466 records. Our analysis examined certain demographic variables such as age, race, income, gender, and education and their effect on the way Americans consume their news, more specifically through the Internet or television. This project made heavy use of SQL within the SAS 9.4 software to import and clean the dataset from the GSS. We also used Photoshop and Affinity designer to create our presentation. Our team won 3rd place and was the only undergraduate team in the competition.

  • SQL, SAS 9.4, Photoshop, Affinity Designer
